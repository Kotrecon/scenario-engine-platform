# EEMUA 191 — Текущее состояние

## Требования EEMUA 191

| Требование                 | Статус      | Описание                                           |
| -------------------------- | ----------- | -------------------------------------------------- |
| Цветовая схема приоритетов | Планируется | Critical=red, High=orange, Medium=yellow, OK=green |
| Без мигания                | Планируется | Только пульсация устройств при аларме              |
| Приоритеты через цвет      | Планируется | Не через форму                                     |
| Читаемость на расстоянии   | Планируется | Контраст, размер шрифта                            |
| KPI: Alarm Rate ≤5/10мин   | Планируется | Метрики через OTel                                 |
| KPI: Standing ≤5%          | Планируется | Метрики через OTel                                 |
| KPI: Stale ≤1%             | Планируется | Метрики через OTel                                 |

## Цветовая схема (планируемая)

| Приоритет | Цвет       | Hex       |
| --------- | ---------- | --------- |
| Critical  | Красный    | `#ef4444` |
| High      | Оранжевый  | `#f97316` |
| Medium    | Жёлтый     | `#eab308` |
| OK        | Зелёный    | `#10b981` |
| Drift     | Фиолетовый | `#a855f7` |
| Info      | Синий      | `#3b82f6` |

## Тёмная тема (планируемая)

| Элемент    | Цвет         | Hex       |
| ---------- | ------------ | --------- |
| Background | Тёмный       | `#0a0c10` |
| Panel      | Чуть светлее | `#111318` |

## UI-компоненты (планируемые)

- Карточки (n8n-style) для триггеров и условий
- Drag & drop для визуального редактора
- Таблицы с фильтрацией и сортировкой
- Модалки для создания/редактирования
- Сайдбары для навигации
- Хедеры с KPI и статусом
